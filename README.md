# DevOps / Cloud Learning Lab

Hands-on, production-style path from zero to job-ready DevOps/Cloud.

## What you'll find here
- 00 Foundations: Linux, Git, Networking
- 01 Containers: Docker, Compose
- 02 CI/CD: GitHub Actions
- 03 Cloud: AWS (core + networking)
- 04 IaC: Terraform
- 05 Kubernetes: EKS, Helm, GitOps (ArgoCD)
- 06 Observability: Prometheus, Grafana, Loki
- 07 Security: IAM, container scanning, K8s policies
- 08 SRE: SLO/SLI, incidents, load testing

## Rule
Everything must be deployable, observable, and reproducible.
